ABOUT

Modern computers run code in parallel. Real-life tasks benefit from parallelism. But... so far, coding games had only sequential instructions! Blockly Par is a platform inspired in the Blockly-Games, from Google. It introduces parallel thinking through games involving day-to-day tasks and parallel coding instructions. There are 3 games currently available: Introduction to Sequential Programming with Blocks, Introduction to Tasks and Parallel Programming.
Blockly Par is specially targeted to freshman students in Computer Science majors. We expect that, at the end of the games, the students feel challenged to explore parallelism in their programs.


This project was developed during a Bachelor thesis in Computer Science at the Federal University of Santa Maria, in Brazil.
