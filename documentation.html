DOCUMENTATION

Each game has 4 levels.

This game introduces the block visual programming and the development environment. The user will use the programming blocks to guide the student on their way home using the Blockly library.
This game introduces the execution of tasks concept. The goal is to guide the student on their way to the classroom. However, in order to successfully go to the classroom and finish each level, the student has to finish the homeworks on the way.
This game introduces parallel programming. The goal is to guide the student on their way to the libray. The challenge to complete at each level is to guide the student to the library to return all the books. The student delivers a book upon arrival at the library. A book is decremented from the list, and the student automatically returns to the starting point of the game and the tasks block execute all the blocks again.
